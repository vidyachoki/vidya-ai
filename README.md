## 🌟 What is Vidya AI?

Vidya AI is a **Universal AI Assistant** that can answer questions on any topic, in any language, understand uploaded files and images, and guide users like an intelligent human mentor.

---

## ✨ Current Features (Phase 1)

### 🧠 Universal Knowledge
- Answers questions on **any topic** — Technology, Business, Health, Finance, History, Science, Cooking, Travel, Sports, and more
- Deep expertise in **Production Support, DevOps, AI/ML, Cybersecurity**
- Career guidance — Resume help, Interview prep, Salary negotiation

### 🌐 Multi-Language Support
- **English, Hindi, Telugu, Tamil, Kannada** and many more
- Responds in the **same language** you write in
- Can teach English with grammar, vocabulary, and practice

### 📁 File Understanding
- Upload **TXT, CSV, JSON, MD** files
- Reads and analyzes entire document
- Answers questions from document content
- Summarizes key information

### 🖼️ Image Analysis
- Upload any image — Vidya describes, reads text (OCR), explains diagrams
- Understands charts, graphs, technical screenshots
- Answers detailed questions about images

### 💬 Conversation Memory
- Remembers entire chat history within a session
- Follow-up questions work naturally
- Context-aware responses

### 🎤 Voice Input
- Click the microphone to speak your question
- Supports English and other languages

### 😊 Friendly Personality
- Warm, caring, motivational like a friend
- Professional when needed, funny when appropriate
- Empathetic support for personal questions

## 🚀 Quick Start

### Option 1 — Use Live Demo
Click the **Live Demo** badge above — works instantly in your browser!

### Option 2 — Run Locally
```bash
# Clone the repository
git clone https://github.com/vidyachoki/vidya-ai.git
cd vidya-ai

# Open in browser
open index.html
# or double-click index.html
```

### Option 3 — Use Your Own API Key
1. Get a free API key at [aistudio.google.com](https://aistudio.google.com/apikey)
2. Open `index.html` in a text editor
3. Replace the API key value with your own key
4. Open in browser

## 💡 How to Use

### Chat
Just type any question and press Enter. Ask anything — no topic limits!

### Upload Files
Click **📎 Attach** and upload:
- Text files (.txt, .csv, .json, .md)
- Images (JPG, PNG, etc.)

Then ask questions about the file!

### Multi-Language
Write in any language — Vidya responds in the same language automatically.

### Quick Topics
Use the **sidebar** to quickly explore topics like AI, Health, Finance, Career, and more.
